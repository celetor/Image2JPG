# ImageConvert

## 支持以下格式转jpg
```commandline
    ext = ['JPG', 'JPEG', 'PNG', 'BMP', 'HEIC', 'HEIF', 'HIF', 'AVIF', 'AV1', 'WEBP']
```

* https://github.com/python-pillow/Pillow
* https://github.com/strukturag/libheif
